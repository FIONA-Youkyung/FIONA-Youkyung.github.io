# SEO 최적화

## SEO란?  
> 검색엔진이 사이트의 모든 컨텐츠를 읽을 수 있도록 사이트 구조를 개선하는 작업 

## SEO 핵심요소 
1. 자연 검색 트래픽 
2. 키워드 랭킹 (브랜드 키워드, Non-brand 키워드 구분하여 측정 / 월단위 측정)
3. 검색엔진 내 인덱싱 된 페이지 수 
4. 자연검색 트래픽 기반 전환 수
